# Window License Related Guid

## Windows Server License extension
## Open Powershell
### Run the following command
`slmgr -dlv`
`slmgr -rearm`
### Restart the Server to take affect

## Windows Activation
## Open Powershell
### Run the following command
`slmgr /skms kms.digiboy.ir`
### Wait for 30 to 60 seconds and then run following command
`slmgr /ato`
